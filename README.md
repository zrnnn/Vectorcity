**Create stunning, customizable vector maps for your laser cutter, 3D printer, or print projects ‚Äì all from a single HTML file!**

## ‚ú® Overview

VectorCity is a lightweight, browser-based tool that leverages OpenStreetMap's Overpass API to generate beautiful, high-detail city maps. Designed with makers in mind, it allows you to easily select an area, customize map features, and export it into versatile formats perfect for physical creation.

## üåü Features

* **Local & Lightweight:** Runs entirely in your web browser from a single `VectorCity.html` file. No server, no installation, just download and open!
* **Laser-Ready SVG Export:** Generate perfectly scalable vector graphics (.svg) optimized for laser cutting, engraving, and CNC machining. Includes intelligent line width scaling.
* **High-Resolution Print PNG Export:** Export crisp .png images at 600 DPI, ideal for art prints, posters, or digital use.
* **Customizable Map Layers:** Toggle and style individual map features like buildings, roads, waterways, and parks. Adjust colors and line thicknesses to create your desired aesthetic, perfect for high-contrast laser projects.
* **Precise Framing:** Define your exact map area and choose from common aspect ratios (A4, 1:1, 16:9, etc.) for perfect composition.
* **Metric Output Control:** Set your desired final output width in millimeters for direct compatibility with your physical projects.

## üöÄ How to Use (Quick Start)

1.  **Download the Tool:**
    * Clone this repository: `git clone https://github.com/zrnnn/Vectorcity.git`
    * Or simply download the `VectorCity.html` file directly from the `main` branch.

2.  **Open in Browser:**
    * Locate the `VectorCity.html` file on your computer.
    * **Double-click** the file to open it in your preferred web browser (Chrome, Firefox, Edge, Safari recommended).

3.  **Create Your Map:**
    * Use the search bar to find your desired location.
    * Adjust the map view to frame your chosen area.
    * Customize the display by toggling layers and changing colors/line widths.
    * Click the "Export" button to save your map as an SVG or PNG.
